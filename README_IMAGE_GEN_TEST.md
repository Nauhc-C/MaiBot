# 图像生成测试说明

## 概述

这是一个独立的图像生成测试脚本，用于测试 `gpt-image-2` API 的生图功能。测试通过后再接入 MaiBot。

## 文件说明

- `test_image_gen.py` - 主测试脚本
- `test_image_gen_config.json` - 配置文件（可选）
- `output/` - 生成的图像输出目录
- `image_gen_test.log` - 测试日志文件

## 功能特点

1. **仅使用 gpt-image-2** - 暂时放弃 nanobanana 源，只测试 image-2
2. **独立测试** - 不依赖 MaiBot 主程序，方便单独调试
3. **详细日志** - 同时输出到控制台和日志文件
4. **多种测试场景**:
   - 基础生图（无参考图）
   - 带参考图生图（场景 + 角色）
   - 自定义提示词测试

## 使用方法

### 1. 安装依赖

测试脚本使用 Python 标准库，无需额外安装依赖。

### 2. 运行测试

```bash
# 在 project/MaiBot 目录下运行
python test_image_gen.py
```

### 3. 查看结果

- 生成的图像保存在 `output/` 目录
- 详细日志保存在 `image_gen_test.log`
- 控制台会显示实时进度和结果摘要

## 参考图说明

### 参考图格式

- 支持格式: PNG, JPG, JPEG, WEBP
- 自动转换为 data URL (base64 编码)
- 支持多参考图（场景 + 角色）

### 参考图目录

脚本会从以下目录查找参考图:

- **场景图**: `D:/Sakiko/assets/场景/`
- **角色图**: `D:/Sakiko/assets/立绘/小祥/`

如果目录不存在或为空，脚本会自动跳过参考图测试。

## 常见问题

### 1. 请求超时

**现象**: `urllib.error.URLError: <urlopen error timed out>`

**解决方案**:
- 检查网络连接
- 增加 timeout 参数（当前 60 秒）
- 减小参考图文件大小

### 2. API 错误

**现象**: `HTTP 401/403/500`

**解决方案**:
- 检查 API Key 是否正确
- 检查 base_url 是否正确
- 查看 `image_gen_test.log` 了解详细错误信息

### 3. 参考图未找到

**现象**: "未找到参考图，跳过此测试"

**解决方案**:
- 确认参考图目录路径正确
- 确认目录中有 PNG 格式的图片文件

### 4. 生成的图像为空

**现象**: "图像生成失败：返回空数据"

**解决方案**:
- 检查 API 响应格式是否变化
- 查看日志中的响应内容
- 尝试简化提示词

## 技术细节

### API 调用流程

1. 构建请求体:
   ```json
   {
     "model": "gpt-image-2",
     "prompt": "提示词",
     "n": 1,
     "response_format": "b64_json",
     "extra_body": {
       "image": ["data:image/png;base64,xxx", "..."]
     }
   }
   ```

2. 发送 POST 请求到 `/images/generations`

3. 解析响应:
   - 优先使用 `b64_json` 字段
   - 备选方案: 从 `url` 字段下载

4. 保存图像到本地

### 参考图处理

1. 读取图片文件（PNG/JPG/WEBP）
2. Base64 编码
3. 包装为 data URL: `data:image/{mime};base64,{encoded}`
4. 添加到请求的 `extra_body.image` 字段

### 错误处理

- 网络错误: 记录详细日志并抛出异常
- HTTP 错误: 记录状态码和响应内容
- 解析错误: 记录响应结构并返回 None
- 文件错误: 记录路径并跳过该文件

## 性能考虑

- **超时设置**: 60 秒（可根据网络情况调整）
- **参考图大小**: 建议单张 < 5MB
- **并发请求**: 测试脚本串行执行，避免速率限制
- **请求间隔**: 测试间等待 2 秒

## 安全注意事项

⚠️ **API Key 安全**:
- 不要将包含真实 API Key 的代码提交到公开仓库
- 生产环境使用环境变量或配置文件管理密钥
- 定期轮换 API Key
